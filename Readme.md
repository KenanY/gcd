# gcd

Greatest common denominator component

## Installation

``` shell
$ component install KenanY/gcd
```

## API

### gcd(a, b)

Calculate the greastest common divisor amongst `a` and `b`.

``` javascript
gcd(1254, 5298);
// => 6
```

## Development

``` shell
$ npm install mocha should
$ make
```
